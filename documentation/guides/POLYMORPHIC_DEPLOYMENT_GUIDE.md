### **Common Issues & Solutions:**

**Issue: Python not found**
```bash

**Issue: Permission denied (Unix systems)**

**Issue: Scripts not found**

```

**Issue: Performance issues**
```bash

# Apply optimizations
python3 scripts/polymorphic_platform_bridge.py setup
```

### **Platform-Specific Troubleshooting:**

**Windows:**
- Ensure PowerShell execution policy allows scripts: `Set-ExecutionPolicy RemoteSigned`
- Use `py` instead of `python3` if Python 3 not found

**macOS:**
- Install Homebrew if package management needed: `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- Use `python3` explicitly (macOS has Python 2 as `python`)

**Linux:**
- Install Python 3 if not available: `sudo apt install python3` (Ubuntu/Debian)
- Ensure pip is available: `sudo apt install python3-pip`

**Cloud IDEs:**
- Verify internet connection for initial setup
- Some cloud IDEs may have resource limits - check with provider
