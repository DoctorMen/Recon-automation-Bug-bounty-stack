## 10. API SECURITY SPECIALIZATION (from hackingapis.pdf)

### **Key API Vulnerabilities to Test:**

#### **1. Authentication Vulnerabilities**
- Weak JWT implementation
- OAuth flow flaws
- API key exposure
- Session fixation
- Token reuse

#### **2. Authorization Vulnerabilities**
- IDOR (Insecure Direct Object References)
- Privilege escalation
- RBAC bypass
- Horizontal vs vertical access control
