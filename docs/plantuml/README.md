# PlantUML Documentation System

This directory contains PlantUML diagrams for the GHOST Security Automation Framework.

## Getting Started

1. Install PlantUML:
   ```bash
   # For Ubuntu/Debian
   sudo apt-get install plantuml
   
   # For MacOS
   brew install plantuml
   ```

2. Generate diagrams:
   ```bash
   cd /path/to/ghost-framework/docs/plantuml
   ./generate_diagrams.sh
   ```

## Diagram Types

- `architecture/`: System architecture diagrams
- `workflows/`: Process and workflow diagrams
- `sequence/`: Sequence diagrams
- `deployment/`: Deployment architecture
