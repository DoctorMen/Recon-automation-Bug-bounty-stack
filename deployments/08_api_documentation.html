Template API

Retrieve all available proposal templates

GET /api/templates?keyword=emergency

POST /api/templates/generate

Generate customized proposal from template

POST /api/templates/generate

GET /api/pricing?template=api&scope=medium

Retrieve performance metrics and analytics

GET /api/metrics?period=month
